Tableau Calculation Assistant

Expect wrong/strange answers
- This is a prototype and POC for demo purposes.
- It'll keep getting better over time.

Best Practices:
- Ask a question in plain English.
- Make your question/insturction relevant to Tableau calculations.
- If you want to reference fields from your dataset, use the field name in brackets. For example, [Sales] or [Profit Ratio].

Limitations:
- Model is still being tuned. Expect incorrect/strange results.
- Model can produce different results for the same prompt. Try re-running the model if you get an unexpected result.
